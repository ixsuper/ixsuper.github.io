# ixsuper.github.io

Personal portfolio and app landing pages for iOS developer **ixsuper** (Ziyad Alsuhaymi).

**Live site:** [https://ixsuper.github.io](https://ixsuper.github.io)

## Apps

- **Echoes: Echoes of the City** -- Capture living memories with photo, ambient sound, weather, and location. Built for iOS 26.
- **Block Blaster Puzzle** -- A satisfying block puzzle game with explosive combos, sensory themes, and haptic feedback.
- **Color Number Match** -- A fun and colorful number matching puzzle game. Available on the [App Store](https://apps.apple.com/app/id6761441678).

## About This Repo

This repository hosts the main portfolio landing page as well as individual privacy policies, terms of use, and support pages for each app, ensuring App Store compliance.
