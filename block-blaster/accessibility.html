Accessibility

Block Blaster Puzzle is designed to be enjoyed by everyone.

We are committed to making Block Blaster Puzzle accessible to all players, regardless of ability. Below are the accessibility features currently supported in the app.

Supported Features

☝ VoiceOver

Full VoiceOver support throughout the app. Every cell on the game grid is labeled with its position and state (e.g., "Row 3, Column 5: Blue block"). Interactive elements include accessibility hints so you always know what actions are available.

🌑 Dark Interface

Block Blaster uses a dark color scheme by default, reducing eye strain and improving readability in low-light environments. All UI elements are designed with sufficient contrast against the dark background.

⚫ Differentiate Without Color Alone

Enable Colorblind Mode in Settings to display unique symbols on each block color. This ensures that block types can be distinguished by shape as well as color, making the game fully playable for players with color vision deficiency.

✨ Reduced Motion

When Reduce Motion is enabled on your device, Block Blaster automatically disables particle effects, screen shake, and other intensive animations. Gameplay remains smooth without any motion that could cause discomfort.

Additional Accessibility Options

🔈 Haptic Feedback Controls

Haptic feedback can be independently toggled on or off in Settings. Players who are sensitive to vibrations or use assistive devices can disable haptics without affecting sound or gameplay.

🌐 Multiple Languages & RTL Support

Block Blaster supports multiple languages including Arabic and Urdu with full right-to-left (RTL) layout. The interface automatically mirrors to match the selected language direction.

👁 Ghost Preview

When dragging a piece onto the grid, a visible ghost preview shows exactly where it will land. The preview uses both color and border styling to indicate valid and invalid placements, providing clear visual feedback before committing a move.

Feedback

We are always working to improve accessibility in Block Blaster. If you encounter any barriers or have suggestions for how we can make the game more accessible, please reach out.
